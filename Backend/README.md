https://documenter.getpostman.com/view/11165950/TzCS6mRU

### Server URL
https://masterofmedicines.herokuapp.com/

### Instructions

Create `.env` file:
```
PORT=3001
SECRET=express_session_secret_here
HOST=host_here
DB=db_name_here
USER=user_here
PASSWORD=password_here 
```


### To Run:

Development: `npm run dev`

Production: `npm start`
